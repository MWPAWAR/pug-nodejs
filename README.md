# Pug NodeJS Demo

This is a demo on how to integrate PUG templating engine with NodeJS. 

## Pre-requisites
1. Install NodeJS
2. Install Yarn

## Installation

```bash
1. Go to Project directory
2. Run `yarn`
3. Rub `yarn start`
```

Now visit `localhost:3000` in browser, the NodeJS app will serve the index file.
